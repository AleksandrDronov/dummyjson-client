# DummyJSON Client

Приложение на React + TypeScript для работы с DummyJSON API.

## Запуск проекта

1. **Установите зависимости:**

   ```bash
   npm install
   ```

2. **Запустите проект в режиме разработки:**

   ```bash
   npm run dev
   ```

   После запуска приложение будет доступно по адресу, который выведет Vite (обычно http://localhost:5173).

3. **Сборка проекта:**

   ```bash
   npm run build
   ```

4. **Просмотр production-сборки:**

   ```bash
   npm run preview
   ```

## Дополнительно

- Для проверки кода используйте:
  ```bash
  npm run lint
  ```
- Проект использует Vite, React 19, TypeScript, ESLint.

## Форматирование кода

Для автоматического форматирования используйте [Prettier](https://prettier.io/):

```bash
npx prettier --write .
```

Настройки находятся в файле `.prettierrc`.
